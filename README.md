## 👨‍💻 Sobre mim

Sou estudante de **Engenharia de Software** com foco total em **desenvolvimento back-end**. Estou aprofundando meus conhecimentos no ecossistema **JavaScript** com direcionamento para **Node.js**, visando a construção de APIs REST robustas, microsserviços escaláveis e soluções eficientes orientadas a serviços.

Possuo experiência prática em versionamento com **Git e GitHub**, além de uma base sólida em lógica de programação, estruturas de dados e conceitos fundamentais de arquitetura de software.

Busco aprimorar minhas habilidades em boas práticas de desenvolvimento (Clean Code, SOLID), arquitetura de software e ferramentas modernas de desenvolvimento, com o objetivo de **ingressar profissionalmente na área**.

> 💡 *"A melhor forma de evoluir é colocando a mão no código."*
