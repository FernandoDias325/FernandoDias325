## 👨‍💻 Sobre mim

Sou estudante de **Engenharia de Software** com foco em **desenvolvimento back-end**. Estou aprofundando meus conhecimentos em **JavaScript** com direcionamento para **Node.js**, visando a construção de APIs REST, aplicações escaláveis e soluções orientadas a serviços.

Possuo experiência em versionamento com **Git e GitHub**, familiaridade com lógica de programação, estruturas de dados, funções, arrays, objetos e conceitos fundamentais do desenvolvimento de software.

Busco aprimorar minhas habilidades em back-end, boas práticas de desenvolvimento, arquitetura de software e ferramentas do ecossistema JavaScript, com o objetivo de **ingressar profissionalmente na área**.

> *"A melhor forma de evoluir é colocando a mão no código."*
